1. Requirement:

- VPS Ubuntu 20.04
- Open public PORT 8088
- Root user

**********************************

2. Setup NodeJS:

sudo apt update
sudo apt install curl ca-certificates -y
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.bashrc
nvm install 18

**********************************

3. Setup project:

npm install
npm install -g pm2
pm2 start app.js -i 0 --name proxy_node_app

**********************************
